







#### **ListResp**[**CourseOut**] 什么意思？

看ListResp类型，意思为ListResp有两个成员变量，data是范型类型，那么在这个例子中data是CourseOut类型。

```
class ListResp(BaseModel, Generic[T]):
    data: Optional[list[T]] = None
    count: int
```
